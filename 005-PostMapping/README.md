# 005 - PostMapping

Spring Boot application with a REST Controller using `@PostMapping` with a `@RequestBody` parameter.  
Tested with Postman.

## Endpoints
- `POST /client` — RequestBody with a Client object `{ id, name, surname }`

## Stack
- Java 26
- Spring Boot 4.x
- Maven

## Testing
- Postman
